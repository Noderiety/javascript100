##Preparation
Install [node.js](http://nodejs.org/)
Install [nvm](https://github.com/creationix/nvm):
```
$ curl https://raw.github.com/creationix/nvm/v0.3.0/install.sh | sh
```

Install latest:
```
$ nvm install v0.10 # Current version is >= 0.10.26
```
